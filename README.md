# Techgram

## Description
An application where developers can blog about web development related topics with the ability to comment on posts and interact with other users

## Built With 
* Javascript
* CSS
* SQL
* Dotenv
* Sequelize
* Bcrypt
* Handlebars
* Express
* Session
